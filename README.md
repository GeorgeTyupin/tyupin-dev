# Tyupin-dev — Portfolio

Персональное портфолио, объединяющее современный бэкенд на Go и динамичный фронтенд.

**Сайт:** [tyupin-dev.ru](https://tyupin-dev.ru)

### Ключевые особенности:
- **Backend**: Написан на Go с использованием роутера `chi` для высокой производительности.
- **Интеграция с GitHub**: Динамическое получение данных о пользователе, проектах и активности через GitHub API.
- **Frontend**: Легкий и быстрый интерфейс на чистом JavaScript и CSS.
- **Инфраструктура**: Полная контейнеризация проекта с помощью Docker и Docker Compose.
- **Deployment**: Настроен Nginx в качестве обратного прокси для стабильной работы.

Простой и эффективный инструмент для демонстрации профессиональных навыков.
